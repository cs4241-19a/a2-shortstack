FEZ EXPLANATION

FEZ is a puzzle game and has numerous in game puzzles that involve various symbols. For example, the game has it's own alphabet which the player is expected to figure out through the rooms in the game which are designed to teach you this language.

One of the game's common puzzles involve tetrominos (which are a common theme in the game). These sequences of tetrominos represent a code that needs to be inputted by the player.

The correct sequence of buttons to solve the puzzle located on the main page is:

UP D UP SPACE D A A DOWN

Try it out! Simply type the code while the focus is on the webpage. If you want to learn why the code can be solved by entering in that combination of buttons, I suggest trying out FEZ!
