About

Why would someone make something like this??

This is an example of a simple node.js app with persistent data store using AWS DynamoDB. The client side JS is responsible for making requests to the node server, and the server handles all communication with AWS services for data storage.

Example JSON Packet

This shows how the information for each note is encoded to be sent between the server and client.

    notes: "this is a note",
    priority: "2",
    title: "Test Title 2",
    unixtime: "1567881747611", // this is a derived field
}
