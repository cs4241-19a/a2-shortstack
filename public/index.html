Random Drawing Generator

Hello and Welcome! The generator will generate random points in a 3D space based on the number of vertices you give it, how you would like the points to be connected, and lastly a name for your art piece. The newly generated drawing would be add to the table below. Click on "View" and the drawing will be drawn in the box to the right.

Here are some animation you can toggle on and off! To turn off the animation, just hit the same key again:

Object moves to the right !
Object moves to the left @
Object moves up #
Object moves down $
Object moves closer to you %
Object moves away from you ^
Object rotates &

Please enter the number of vertices (1 - 50).

Please enter the number of triangles (1 - 50).

Please enter a name for your to be generated art piece.

Please use a brower that supports the "canvas" tag.
